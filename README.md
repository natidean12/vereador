# 📦 Portfólio Institucional – Coronel Marcelo Vieira Salles
# 📌 Sobre o projeto

O Portfólio Institucional do Coronel Marcelo Vieira Salles é um website desenvolvido para apresentar sua trajetória profissional, experiência na Polícia Militar, atuação no poder público, principais projetos legislativos, leis aprovadas e canais oficiais de contato. O projeto foi criado com foco em transmitir profissionalismo, credibilidade e transparência, oferecendo uma navegação moderna e intuitiva para qualquer visitante.

O site reúne informações completas sobre sua carreira militar, passagem pela administração pública e mandato como vereador, utilizando uma interface elegante com efeitos visuais modernos e organização em seções temáticas. Além disso, conta com recursos interativos como chatbot institucional, linha do tempo profissional, acordeões para apresentação dos projetos, carrossel de imagens, formulário de contato e integração com o Google Maps.

Toda a interface foi desenvolvida utilizando um design sofisticado, responsivo e otimizado para computadores, tablets e smartphones, garantindo excelente desempenho e experiência de navegação em qualquer dispositivo.

# 🎯 Objetivo

Desenvolver um portfólio institucional moderno, elegante e totalmente responsivo para apresentar a trajetória profissional, projetos, experiência pública e principais realizações do Coronel Marcelo Vieira Salles, oferecendo uma navegação intuitiva, recursos interativos e uma comunicação clara com os visitantes.
